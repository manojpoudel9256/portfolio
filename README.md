セットアップ方法

リポジトリをクローン：

git clone https://github.com/manojpoudel9256/portfolio.git


プロジェクトフォルダに移動：

cd portfolio


ブラウザで index.html を開く。

このポートフォリオは静的サイトですので、サーバー構築は不要です。
